Catálogo importado

Ya está presente en el repositorio dentro de catalogo/, pero esta carpeta contiene una aplicación Next.js. Eso significa que no se muestra automáticamente desde la página principal estática.

Para ver el catálogo en funcionamiento, necesitás desplegar la app Next.js de catalogo/ en un servidor que soporte Node, o convertirlo a un sitio estático exportado.

Si querés, puedo ayudarte a:

  • crear un build estático para servirlo en este mismo dominio
  • integrar solo el catálogo en la página principal
  • configurar un despliegue separado con Vercel/Render
